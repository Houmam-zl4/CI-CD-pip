# Projet Jenkins + Docker

Ce projet consiste à automatiser la création et le déploiement d'une application web simple avec Jenkins et Docker.

## Étapes réalisées :
- Installation de Jenkins et configuration des jobs pour l'intégration continue.
- Création d'une image Docker avec un serveur Nginx.
- Configuration d'un pipeline CI/CD pour construire, publier et déployer l'image Docker.
- Modification de la page `index.html` et tests automatiques du pipeline.
